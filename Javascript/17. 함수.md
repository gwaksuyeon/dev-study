### 17. 함수

#### spread 문법

- 배열을 인수목록으로 확장

```javascript
let arr =  [3, 5, 1];

console.log(Math.max(arr)); // NaN

// 숫자 목록을 인수로 받아 오류
// console.log(Math.max(arr[0], arr[1]...))
// 배열의 길이도 알 수 없고 코드가 지저분해짐
```

```javascript
let arr = [3, 5, 1];

console.log(Math.max(...arr)); // 5
```

- 객체 여러개 전달 가능, 평범한 값과 혼합해서 사용 가능, 배열 합칠때 사용가능
